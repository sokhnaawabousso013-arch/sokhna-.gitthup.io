CV Interactif — Sokhna Awa Bousso
Projet académique réalisé dans le cadre du module Développement Web — Licence 2 Informatique, Université Iba Der Thiam de Thiès.
Technologies utilisées
•HTML5 — Structure sémantique
•CSS3 — Design responsive (Grid, Flexbox, variables CSS, media queries)
•JavaScript Vanilla — Interactivité (sans bibliothèque externe)
Fonctionnalités
•Mode sombre / clair avec sauvegarde localStorage
•Menu hamburger responsive
•Animation des barres de compétences au scroll (IntersectionObserver)
•Formulaire de contact avec validation côté client
•Bouton retour en haut
•Navigation fluide avec ancres (scroll-behavior: smooth)
